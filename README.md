# Dotfiles

Personal dotfiles using [Dotbot](https://github.com/anishathalye/dotbot) for cross-platform configuration management.

## Quick Start

```bash
# Clone repository
git clone --recurse-submodules https://github.com/dotMeeko/dotfiles.git ~/.dotfiles

# Install dotfiles
cd ~/.dotfiles
./install      # Linux/macOS
.\install.ps1  # Windows
```
